## 备考这件事

软考中项的知识面很宽，但真正让人犯难的不是"不会"，而是**不知道从哪里开始**。

30 天计划解决的就是这个问题：不用想今天该刷什么，打开就知道。

## 克制的设计

我刻意砍掉了很多"可以有"的功能：

- 没有账号系统，数据存本地
- 没有学习报告，只有当天进度
- 没有社区排名，只有你和题目

因为备考需要的不是更多信息，而是**更少干扰**。

```typescript
// 今日任务：固定 20 题，基于当前 day 偏移取题
function getTodayQuestions(day: number, bank: Question[]) {
  const start = ((day - 1) * 20) % bank.length;
  return bank.slice(start, start + 20);
}
```
