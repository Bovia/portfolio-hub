## 为什么做这个

备考金融证书那段时间，我翻遍了市面上的 App——广告弹窗、会员墙、加载缓慢。我只想安静地刷题，找不到这样一个地方，于是自己做了一个。

没有账号，没有付费，打开就能练。

## 它长什么样

界面只有一件事：**下一题**。

每道题答完，立刻告诉你对错，配上简洁的解析。答错的题会在之后悄悄重新出现，不需要你手动管理错题本。进度存在本地，下次打开接着来。

```typescript
// 错题复现逻辑：简单但够用
function shouldRepeat(q: Question): boolean {
  return q.wrongCount > 0 && Math.random() < 0.4;
}
```

## 一个小决定

我特意没做"排行榜"或"连续答题天数"——那些东西会让人焦虑，而不是专注。

这个工具只服务于一件事：**让你在考试那天多答对几题**。
