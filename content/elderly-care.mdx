## 一个真实的起点

有一次我父亲给我发消息，说他按了半天那个"小字儿按钮"，怎么也没反应。

我看了下他说的界面——按钮是 32px 的，灰色文字在浅灰背景上，对比度勉强够用。对我来说完全正常，对他来说是一道屏障。

这个项目从那天开始有了意义。

## 设计的核心不是功能，是信任感

老年用户不怕功能少，怕用错了"搞坏了手机"。

所以每个操作都有二次确认，每个按钮都足够大，每个重要动作都有声音反馈。界面不追求"好看"，追求**每次用完都没有挫败感**。

```typescript
// 字号档位，用户可自由选择
const fontScale = {
  comfortable: 1,
  large: 1.25,
  xlarge: 1.5,
};
```

## 做完之后

父亲现在用得不错。他说"这个我会用"，这六个字比任何用户评分都重要。

这个项目让我重新理解了"易用性"——它不是加一个无障碍标签，而是真的去想象那个坐在屏幕前有点犹豫的人。
